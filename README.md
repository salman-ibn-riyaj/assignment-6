## 📖 About The Project

**Assignment-6** is a sleek digital product store built with React. Users can browse a curated collection of premium digital tools, add them to cart, and enjoy a smooth shopping experience — all powered by local JSON data and a modern UI stack.

---

## ✨ Features

- ☁️ **1 TB Storage** — Ample space to store all your digital assets securely in the cloud.
- 🔒 **End-to-End Encryption** — Your data is fully protected with industry-grade encryption at every step.
- 📱 **5 Device Sync** — Stay in sync across up to 5 devices seamlessly in real time.

---

## 🚀 Tech Stack

 ⚛️ React.js 
 🎨 Tailwind CSS 
 🌼 DaisyUI 
 ⚡ JavaScript ES6+ 
 🔔 React-Toastify
 📦 JSON 

---

## 🛠️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/assignment-6.git

# Navigate into the project
cd assignment-6

# Install dependencies
npm install

# Start the development server
npm run dev
```
